 - 하지만, 작성만 편했을 뿐, 사실 실시간으로 jsp가 톰켓과 같은 웹컨테이너에 의해 .java서블릿으로
   변환되고 컴파일되어 진다. (시점은? jsp를 서버에 배포 후 최초의 클라이언트 요청이 있을 떄..)

 - 내장 객체 (Built-in Object)
    서블릿을 직접 사용하지 않으므로, 서블릿에서 사용해왔던 각종 객체들의 레퍼런스 변수명을
    javaEE 에서 정해 놓은, 즉 이미 정해진 명칭의 객체들을 가르켜 내장객체라 한다.
    요청객체 : request 객체
    응답객체 : response 객체
    출력스트림 : out 객체

    service(HttpservletRequest request, HttpServletResponse response){
    	printWriter pw = response.getWriter();
    }
